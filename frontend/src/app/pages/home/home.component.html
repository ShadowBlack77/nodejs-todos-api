Welcome in NodeJS Todos API!

For this project, I built a very simple Frontend using Angular to visually represent how this API works. I included here only the things required for operation, such as

  • Login
  • Registration
  • Logout
  • Cookies forwarding
  • Guards
  • Interceptor with Automatic JWT refresh
  • Todos CRUD

Therefore, this frontend is a brilliant base for expanding this project, or combining it with my other projects related to NodeJS or NestJS API!

Some Frontend Features

  • installed TailwindCSS
  • installed GSAP
